# rogue-storybook
A Roguelike dungeon crawler in JavaScript.

Copyright Jeff Lassahn 2026, see LICENSE for details.

This game was inspired by (and borrows many mechanics from) _Rogue Fable III_
by Justin Wang.  After enjoying that game, I found myself wanting something
like it which had the same sense of exploring different regions, and the
traditional Roguelike random variation in locations, but with more of a sense
of story.

So I'm developing this to allow a single game engine to provide a variety of
different stories, other quests, things not just the traditional "find the
Goblet of Yendor".

If you want to try out the game in its current state, you can run it directly
from GitHub by going to
https://jlassahn.github.io/rogue-storybook/examples/fullscreen.html
or if you want to download it and embed it into a web page, see
`docs/embedding.md`.
